1) Enable PowerShell (if needed)
	a) Run PowerShell with administrator privileges
	b) Run the command: Set-ExecutionPolicy Bypass -Scope CurrentUser -Force

2) Right-click the file "Update_Or_Install_Startup_Scripts.ps1" and click "Run with PowerShell"

3) Install ImageMagic:
	Install link: https://imagemagick.org/script/download.php#windows
	Note: You must select "Install development headers and libraries for C and C++" on the "Select Additional Tasks" screen.
	Reference Document: https://docs.wand-py.org/en/0.6.10/guide/install.html
